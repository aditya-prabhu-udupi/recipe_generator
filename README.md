# Recipe Generator

AI Recipe Generator is a Flask-based web application that generates recipe ideas based on ingredients provided by the user.  
This project was built to understand backend development, API integration, and building a complete end-to-end web application.

Built with **Python (Flask)** and deployed on **Render**.

---
## Features

- Users can input available ingredients
- Generates creative recipe suggestions using AI
- Clean and simple user interface
- Handles user input and API responses efficiently

---
## Tech Stack

- Python
- HTML
- CSS

## How to Run the Project Locally

### 1. Clone the repository
git clone https://github.com/aditya-prabhu-udupi/recipe_generator.git

### 2. Navigate to the project directory
cd recipe_generator

### 3. Create a virtual environment
python -m venv venv

### 4. Activate the virtual environment
- venv\Scripts\activate (Windows)
- source venv/bin/activate (Linux/MacOS)

### 5. Install dependencies
pip install -r requirements.txt

### 6. Set up OpenAI API key
- setx OPENAI_API_KEY "your_api_key_here" (Windows)
- export OPENAI_API_KEY="your_api_key_here" (Linux/MaxOS)

### 7. Run the application
python app.py

### 8. Open in browser
http://127.0.0.1:5000/
